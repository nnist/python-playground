What is this?

This is a tool which finds easily typeable words for use in passphrases.

Why should I use a passphrase?

  • Generally stronger than a regular password
  • Easy to remember multiple passphrases
  • Easier to type
  • Unlikely to be in existing password databases

Passphrase tips

  • Intentionally mis-spell words to increase uniqueness
  • Turn letters around, i.e. deep-fried = freep-died
  • More characters = more secure
  • Make up your own words
